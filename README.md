# Omi Map Notes

Drop notes on the map automagically using transcription from the [Omi](https://www.omi.me/) device.
Winner of the Omi hackathon in Berkeley.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
